Part 2: STIKHOI Poetic Styling

HTML Cheat Sheet

HTML tag Function
<h2></h2> Header with number indicates level from highest [h1] to lowest [h4]
<ol></ol> Ordered List
<li></li> List Item: Marks the individual lines in the poem.
class="verse" Designates the way to style this ordered list
<span></span> Span: Section of text within other content. Here a way of introducing and delimiting class attribute
class="tab" References tab style in the CSS to add tab to the span
lang="grc" References language style in CSS. Here "grc" stands for Greek and sets appropriate polytonic font
<br> Break: forces a new line
<i></i> Italics
class="verse couplets" start="1" Designates the style for the ordered list as verse and couplets (producing indentation on the pentameter).
start="1" Starts the ordered list on verse 1--An important designation when designating which line to indent as the HTML/CSS obviously are not scanning the poetry for you.
